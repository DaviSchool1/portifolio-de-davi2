Galeria de Arte Digital

Este projeto reúne uma série de ilustrações digitais criadas ao longo de 2024, explorando diferentes técnicas e estilos visuais. Cada peça foi desenvolvida com atenção às tendências atuais de design, utilizando paletas de cores vibrantes e contrastantes.

As obras foram criadas usando uma combinação de ferramentas digitais, incluindo Adobe Illustrator para os trabalhos vetoriais e Aseprite para as pixel arts. O projeto também inclui experimentos com texturas e padrões geométricos, buscando criar composições visualmente impactantes.

Além do aspecto técnico, cada ilustração carrega um conceito ou mensagem pessoal, muitas vezes relacionada a temas como identidade juvenil, tecnologia e conexões humanas na era digital.

"A arte digital é minha forma de traduzir ideias abstratas em algo tangível e compartilhável."

Animação e Efeitos Visuais

Coleção de projetos de animação desenvolvidos para plataformas de redes sociais, com foco em conteúdo para Instagram e TikTok. Cada animação foi planejada para contar uma micro-história em poucos segundos, desafio que exigiu síntese criativa e domínio técnico.

As técnicas utilizadas incluem keyframe animation, morphing, motion tracking e composição de efeitos visuais. Os softwares principais foram Adobe After Effects para as animações mais complexas e CapCut para edições rápidas e adaptáveis às tendências das plataformas.

Este projeto também inclui uma série de templates criativos que podem ser personalizados para diferentes necessidades, demonstrando minha capacidade de criar soluções visuais versáteis.

"No mundo das redes sociais, cada frame conta - e eu me dedico a fazer cada um deles valer a pena."

Experimentos com JavaScript

Série de pequenos projetos de programação que exploram o potencial criativo do JavaScript. Desde jogos simples baseados em navegador até visualizações de dados interativas, cada experimento foi uma oportunidade de aprender novas técnicas enquanto produzia algo visualmente interessante.

Destaques incluem um gerador de arte procedural que cria composições abstratas únicas a cada execução, um simulador de partículas que reage à interação do usuário e uma ferramenta de colorização de paletas que ajuda designers a encontrar combinações harmoniosas.

Estes projetos foram desenvolvidos principalmente com JavaScript vanilla, com alguns experimentos utilizando bibliotecas como p5.js para gráficos e Three.js para elementos 3D. O foco sempre foi manter o código limpo e documentado, mesmo em projetos pequenos.

"Programação é outra forma de arte - onde a lógica encontra a criatividade para criar experiências únicas."
